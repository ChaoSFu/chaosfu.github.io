📊 指标说明

涨幅（ret）
板块当日涨跌幅 = 收盘价 / 前收盘价 - 1

人气（pop）
资金关注度 = Z(成交额放量) + 0.5 × Z(上涨家数)
· 成交额放量 = 今日成交额 / 近5日平均成交额
· 上涨家数 = 板块内上涨个股数量

持续性（persistence）
0-3分，评估板块强势的持续时间
· 1日涨幅 > 0 得1分
· 近3日累计涨幅 > 0 得1分
· 近5日累计涨幅 > 0 得1分

分歧（dispersion）
板块内个股涨幅的标准差，衡量走势一致性
· 分歧度越小 → 个股走势越一致 → 板块得分越高
· 分歧度越大 → 个股分化严重 → 板块得分越低

综合分（score）
多因子加权得分 = Z(涨幅) + Z(人气) + 0.5×Z(一致性)
用途：购买推荐依据（STRONG_BUY/BUY/WATCH/AVOID）
· Z() 表示 Z-score 标准化，将不同指标归一到同一尺度
· 注意：榜单按涨幅排序，而非综合评分
